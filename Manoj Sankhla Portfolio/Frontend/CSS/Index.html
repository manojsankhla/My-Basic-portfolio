HTML

HTML5 is the latest version of HyperText Markup Language (HTML), used for structuring web content. It introduces new elements, attributes, and APIs to enhance web development.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to design and control the appearance of web pages. It allows developers to style HTML elements and create visually appealing layouts.

Java Script

JavaScript (JS) is a powerful, lightweight, and dynamic programming language primarily used for web development. It enables interactive and responsive user experiences.

Jquey

jQuery is a fast, lightweight, and feature-rich JavaScript library designed to simplify HTML DOM manipulation, event handling, animations, and AJAX interactions.

My SQL

MySQL is an open-source relational database management system (RDBMS) developed by Oracle Corporation. It is widely used for web applications, data storage, and enterprise solutions.

Boostrap

Bootstrap is a popular open-source CSS framework used for building responsive and mobile-first websites quickly. Developed by Twitter, it includes ready-to-use components, utilities, and JavaScript plugins..
